## Theming
Command | Description
--- | ---

## Plugins
Command | Description
--- | --- 
`wp plugin list` | Get list of installed plugins
`wp plugin activate my_plugin` | Activate plugin my_plugin
`wp plugin deactivate my_plugin` | Deactivate plugin my_plugin
